2.分支结构

1.if 语句

用来判断
有三种格式

1.判断一种情况

java

关系式为true则执行，false则跳过

2.判断两种情况

java

关系式为true执行语句体1，false执行语句体2

3.判断多种情况

java

switch 语句

进行匹配

1.语句格式

case只能是值

2.其他知识点

1.default的位置和省略
1.位置

可以放在任意位置，不会影响正常功能

2.省略

可以省略，但是省略后没有备选输出

2.case穿透

没有break会导致 case穿透
这是因为在执行过程中，发现break才会停止进程，
而没有break会一直执行，直到发现break或者结束，会导致打印全部语句体。
如果多个case的语句体重复，可以利用case穿透简化代码
