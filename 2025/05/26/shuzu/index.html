数组

一种容器，能够存储同种数据类型的多个值
需要结合隐式转换考虑

数组的定义

int [] a
（起名字）

数组的初始化

在内存中为数组容器开辟空间，并将数据存入数组的过程

  • 静态初始化

创建完长度为3，不能再改变

  • 动态初始化

String arr[] = new String[50];
虚拟机会定义一个默认初始化值
整数：0
小数：0.0
字符：/n0000（空格）
布尔：false
引用数据类型：null

数组元素访问

直接打印数组名会出现地址值

  • 获取：
    变量 = 数组名[索引] //从0开始
    直接打印也可以
  • 存储
    数组名[] = 数据/变量 //

数组的遍历

将所有内容取出来的过程
.length :获取数组长度

  • 求和
  • 交换数据
  • 打乱数据

数组的内存

java 内存分配

  • 栈：方法运行时使用的内存
  • 堆：储存对象或数组，new创建的都在这里
  • 方法区：存储可以运行的class文件
  • 本地方法栈：虚拟机jvm使用操作系统时使用
  • 寄存器：给cpu使用

当两个数组指向同一个空间时，其中一个数组改变了小空间的值，那么其他事数组访问的都是改变后的结果
